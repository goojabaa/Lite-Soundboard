# 🇬🇧 English

## 📌 Overview

Lite Soundboard is a lightweight Windows soundboard application designed for streamers and content creators.  
It allows you to trigger sound effects instantly using customizable hotkeys.

From the interface shown:

- Add / Remove sounds
- Edit sound names
- Play / Stop manually
- Assign shortcut keys (Record 2 Numpad keys)
- Adjust master volume
- Clear assigned keys
- Choose preset styles (Default dropdown)
- Customize button color
- Live instruction guide on screen

If no `sounds.json` file exists, the app prompts you to add sounds.

---

## ⚙️ Features

- 🎛 Clean dark modern UI
- ⚡ Low resource usage
- 🔢 2-key hotkey system (Numpad only)
- 🎹 Record keys directly from keyboard
- 🔊 Master volume slider
- 🗂 Dynamic sound management
- 🧹 Clear shortcut keys
- 🎨 Custom color settings
- 📁 JSON configuration file support
- 🪟 Windows desktop application

---

## 🎮 How Hotkeys Work

1. Click **Record Keys**
2. Press any **2 keys on the Numpad**
3. The combination is saved
4. Press the combo anytime to trigger the sound

Example:  
Press `1` then `2` on Numpad → plays assigned sound.

---

## 📂 Configuration

The app reads/writes `sounds.json` in the root directory.

# 🇹🇭 ภาษาไทย

## 📌 ภาพรวม

Lite Soundboard คือโปรแกรม Soundboard สำหรับ Windows  
ออกแบบมาให้ใช้งานง่าย เหมาะสำหรับสตรีมเมอร์และผู้สร้างคอนเทนต์

จากหน้าตาโปรแกรมในภาพ สามารถทำได้ดังนี้:

- เพิ่มเสียง (Add)
- ลบเสียง (Remove)
- แก้ไขชื่อเสียง (Edit Name)
- กดเล่น / หยุดเสียง (Play / Stop)
- บันทึกคีย์ลัด (Record Keys)
- ล้างคีย์ลัด (Clear Keys)
- ปรับระดับเสียงรวม (Volume)
- เลือกธีม/สไตล์ (Default Dropdown)
- ตั้งค่าสีปุ่ม (Set Color)
- มีคำแนะนำการใช้งานแสดงบนหน้าจอ
- แจ้งเตือนเมื่อไม่มีไฟล์ `sounds.json`

## 🎮 快捷键说明

1. 点击 **Record Keys**
2. 按下数字小键盘上的两个数字
3. 组合会被保存
4. 之后按该组合即可触发声音

例如：  
按 `1` 然后 `2` → 播放对应声音
